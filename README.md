Backend-сервис реализует систему регистрации, авторизации и аутентификации. Реализована система управления доступа к ресурсом на основании выданных прав доступа

Приложение реализовано с помощь FastAPI, SQLAlchemy(async) и JWT

---


## Основной функционал

- Регистрация и авторизация пользователей (JWT)
- Обновление активного токена
- Вход и выход из аккаунта
- Роли и уровни доступа
- Проверка прав доступа через dependencies

---


## запуск

1. Клонирование репозитория 
```bash
git clone https://github.com/Mr-Dron/authorization-system
```
2. Создать .env файл и заполнить данными
```bash
cp .env.template .env
```
3. Запустить контейнеры
```bash
docker compose up --build
```

- приложение: http://127.0.0.1:8000
- swagger документация: http://127.0.0.1:8000/docs


---

## Система разграничения прав доступа

Система прав доступа реализована на уровне предустановленных ролей и уровенями доступа. Для масштабирования роли могут создаваться динамически, а права доступа назначаются персонально для каждой роли
